# Welcome

Start writing your first blog post!

## Getting Started

Add markdown files to this directory and run:

```bash
php artisan content:sync
```

## Frontmatter

Each post should have frontmatter:

```yaml
---
title: My Post Title
category: Technology
tags: laravel, php, web-dev
excerpt: A short description
published_at: 2026-02-06
is_featured: true
---

Your content here...
```
